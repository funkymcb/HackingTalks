• ALL User Data is untrusted!
  • NEVER put untrusted Data in HTML Document
  • if REALLY needed, HTML/Js/CSS Encode Data before
  • URL Encode Request URLs (parameters)
  • Quote untrusted Data in HTML attributes
      • <div attr="...ENCODE UNTRUSTED DATA BEFORE PUTTING HERE...">
    • Quoted attributes can only be broken out of with " or '
    • Unquoted attributes with: [space] % * + , - / ; < = > ^ and |
  • Don't try it yourself! Use libs!
