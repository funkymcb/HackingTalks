1. Stored XSS
    • Also referred to as Persistent XSS
    • Scripts are stored on the Server (Databases, Forums, "About Me")
    • Triggered by Users, Applications or Backdoors
    • Special form: Blind XSS
      • Malicious Script is being sent to Backend/Admin
      • For example via Review / Feedback Form
      • Once Form is opened, script will be executed
